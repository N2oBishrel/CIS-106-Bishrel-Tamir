This program prompts the user for the name of their dog and its age in human years. 
It then calculates and displays the age of their dog in dog years, 
based on the popular myth that one human year equals seven dog years. 

Main program
    Get dog's name
    Get dog's age
    Calculate dog years
    Display Result


Get Dog's Name
    Parameters:
        None
    Process:
        Prompt for name
        Get name
    Return value:
        Name

Get Dog's Age
    Parameters:
        None
    Process:
        Prompt for age
        Get age
    Return value:
        Age

Calculate Dog Years
    Parameters:
        age
    Process:
        Calculate dog years (age * 7)
    Return value:
        Dog years

Display Result
    Parameters:
        Name
        Dog years
    Process:
        Display name and dog Years
    Return value:
        None
